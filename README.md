# Angular First App - Homes Housing Application

A hands-on learning project built to master Angular fundamentals. This application demonstrates core Angular concepts through a practical housing search interface, rebuilt from the official [Angular First App tutorial](https://angular.dev/tutorials/first-app).

## Learning Objectives

This project is a way for me to begin learning modern frontend development with Angular. Through rebuilding this application, I've developed practical experience with:

- **Component Architecture** - Creating and composing reusable components
- **Template Syntax** - Angular's declarative template system including interpolation, property binding, and event binding
- **Dependency Injection** - Understanding and implementing Angular's DI system
- **Routing** - Configuring navigation between different views using Angular Router
- **Services** - Building injectable services for shared functionality
- **Reactive Forms** - Implementing form handling and validation
- **HTTP Client** - Making mock API calls to external services via json-server
- **TypeScript** - Leveraging static typing for maintainable code
- **Angular CLI** - Using the command-line interface for project scaffolding and development

## Key Features Implemented

- Property listing display with dynamic data binding
- Housing location search and filtering functionality
- Detailed property view with routing
- Contact form
- Responsive design
- Integration with mock API data using json-server

## Technology Stack

- **Angular 17+** - Modern framework features including standalone components
- **TypeScript** - Strict mode enabled
- **RxJS** - Reactive programming patterns
- **Angular Router** - Client-side navigation
- **CSS** - Component-scoped styling
- **Angular CLI** - Development workflow automation

## Getting Started

### Prerequisites

- Node.js (v18.x or later)
- npm (v9.x or later)

### Installation

```bash
# Clone the repository
git clone [[repository-url](https://github.com/JJenk89/angular_tutorial)]

# Navigate to project directory
cd angular-tutorial

# Install dependencies
npm install

# Start development server
ng serve
